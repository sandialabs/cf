PCMM Planning

PCMM Planning tool defines questions and fields to detail the PCMM elements or/and subelements.

To start editing PCMM planning, please select a role in the top-right corner drop-down.

Actions:

  • To manage planning click on an element in the PCMM wheel.
  • To add a tag click on "New Tag" button.
  • To select a tag select it into the "Tag" drop-down.
  • To manage tag click on "Manage Tag" button. This will display a popup with all the existing tags. You will be able to rename or delete the tags.
