Preferences

It is possible to configure the cf plugin with the preference page under Eclipse Preferences, then Credibility Framework. Two options are available:

  • Display Credibility Framework Version: If set to true (default value), this option displays the version number next to the opened credibility file name:
  • Query file path (experimental): This option references the query file path (default empty). If a valid query file is filled out, a new part is shown on top of the QoI table on "PIRT, Phenomena > QoI Home View" view. This query file allows the user to query the PIRT database.
