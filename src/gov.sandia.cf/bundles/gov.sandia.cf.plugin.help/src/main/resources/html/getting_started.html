Getting Started

Create New Credibility Project

The user must be able to create a new Credibility assessment from an existing modSim project. PIRT can be associated with any point of a SAW project. For example, a project may have multiple ModSim models within a deeply hierarchical folder structure. Conceptually an analyst would navigate to a particular model to create a PIRT assessment. Right-clicking on the desired project, the plugin should provide a credibility creation wizard. The plugin must use SAW current interface to create a new project.

During creation, the interface should ask the user to complete the following information: - "new credibility file" name (file extension will be *.cf)

  • "credibility configuration file" location (yaml file generated)

Just after creation, the new project should be opened in a new editor.

Credibility configuration should be stored in the selected project itself. Once created, the configuration is not supposed to change.

The new credibility project informations are stored in a database.

Open Existing Credibility Project

To open an existing credibility project, the user should double-click on the *.cf file under the desired modSim project in the Project Explorer. The plugin will open a new editor and go on plugin Home View.

Save Credibility Editor
