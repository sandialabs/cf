Uncertainty Inventory

Uncertainty Inventory Tool first goal is to enumerate, describe and associate references for known uncertainties.

Uncertainty Inventory content persists across sessions in *.cf file.

Uncertainty Inventory View

This view lists all the uncertainties associated to the current ModSim. It is possible to add, update, open, delete a new Group of uncertainty or an Uncertainty.

Uncertainty Table

By default the table has ten columns: - Uncertainty group: the group of the uncertainty - Variable Name: the name of the uncertainty variable - Description: a description of the uncertainty - Type: the type of the uncertainty - Characterization: The uncertainty characterization - Parameters: The list of parameters - Model Feature: is it a model feature? - Comments - Feasible Physical Range - References: the evidence link.

Table Events

  • Double-clicking on a row will open the uncertainty details view.

Buttons

Add Uncertainty Group

This button adds a new uncertainty group. It throws a new group creation wizard.

The user can enter a name for the new group. The name can not be null.

Add Uncertainty

This button adds a new Uncertainty. It throws a new Uncertainty creation wizard.

The user can enter all the parameters needed for the uncertainty and associate it to an existing group.

Edit

This button allow to update the selected uncertainty or group. A new wizard is opened.

Delete

This button permits to delete the selected uncertainty or group.

A deletion confirmation dialog should be opened on click.

If the user confirms deletion, the selected uncertainty or group is deleted with all the associated elements.

Back

"Back" button permits to return to cf-plugin Home View.
